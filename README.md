# Student Portfolio Hub

## Overview

Student Portfolio Hub is a full-stack web application
designed to showcase skills, achievements, and projects.

- Technical skills section
- Dynamic project display
- Backend API services
- MySQL database integration

## Technologies

- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
